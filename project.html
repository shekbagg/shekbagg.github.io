IVLE Desktop Sync (Sept 2011 - Dec 2014) Website

Personal Project
Desktop App Win + Mac Java PHP MySQL

IVLE Desktop allows National University of Singapore students to keep a folder on their desktop in sync with the course folder on NUS Course Management system(IVLE). Whenever a professor adds new files(lecture notes, tutroials, exam papers), the desktop app downloads it and organizes it inside a folder on your desktop. Apart from syncing files it also shows new course announcements and forum posts. The app uses IVLE API to pull its data.

The app currently has 4000+ users and has recieved a buyout offer from the National University of Singapore.


Taxi Demand Prediction (Aug 2014 - Apr 2015)

School Project
Linear Regression Pearson Correlation Ruby on Rails Google Charts API

The aim of this project is to be able to predict the demand in all parts of Singapore, given the demand data of a subset of Singapore. This is to reduce taxi waiting time and help cab drivers optimize their placement. The training data consists of 3 months of pickup demand data retrieved from SMRT (Singapore's major transport operator). I am finding demand correlations between different regions for a particle day and time period. Data is visualized using the Google Charts inside a Ruby on Rails app.

My supervisors are Assoc. Prof Vincent Tan from NUS and Afian Anwar from MIT.


SlingPrint (Jan 2013 - May 2013) Website

Freelance Project
Web App PHP PayPal API MySQL

SlingPrint is a website that sells t-shirts in a crowdfunded manner in Singapore. Users can create campaigns by designing shirts and if the campaign reaches a certain threshhold, the shirts are printed and shipped. If it doesn't reach this number, then everyone gets a refund. I co-built the website while primarily focussing on implementing a robust payments system. It uses PayPal's API to process payments and automatically refund customers on campaign cancellation.


Arctic Books (Jul 2012 - Sep 2012)

Personal Project
PHP Javascript

Arctic Books was a used textbook trading website which scraped data from NUS's used textbook forum, and then origanized the data to be dsiplayed like a traditonal ecommerce website. It allowed users to connect to their NUS account and see textbooks for the courses they were taking. It also allowed users to subscribe to books for a certain course, allowing them to be notified the moment it became available.

During peak sales periods(start of semester) the site recieved 10000+ hits a day.
