IVLE Desktop Sync (Sept 2011 - present) Website

Personal Project
Desktop App Win + Mac Java PHP MySQL

IVLE Desktop allows National University of Singapore students to keep a folder on their desktop in sync with the course folder on NUS Course Management system(IVLE). Whenever a professor adds new files(lecture notes, tutroials, exam papers), the desktop app downloads it and organizes it inside a folder on your desktop. Apart from syncing files it also shows new course announcements and forum posts. The app uses IVLE API to pull its data.

The app currently has 4000+ users and has recieved a buyout offer from the National University of Singapore.


Taxi Demand Prediction (Aug 2014 - present)

School Project
Linear Regression Pearson Correlation Ruby on Rails Google Charts API

The aim of this project is to be able to predict the demand in all parts of Singapore, given the demand data of a subset of Singapore. This is to reduce taxi waiting time and help cab drivers optimize their placement. The training data consists of 3 months of pickup demand data retrieved from SMRT (Singapore's major transport operator). I am finding demand correlations between different regions for a particle day and time period. Data is visualized using the Google Charts inside a Ruby on Rails app.

My supervisors are Assoc. Prof Vincent Tan from NUS and Afian Anwar from MIT. The project is under the umbrella of TaxiSG.


Run Mapper (Sep 2014 - present) Website Github

Personal Project
Web App Javascript Ruby on Rails Google Maps V3 API Facebook Graph API

Run Mapper is a project aimed at helping runners find running trails around them. In the future it hopes a become a social platform for runners with the ability to create running meetups and invite their friends. Route creation uses Google Maps V3 API to allow users to plot routes, find route distance and create elevation profiles.

The app is currently hosted on Heroku.
